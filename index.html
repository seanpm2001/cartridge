Welcome to the Cartridge project. Cartridge is a shopping cart application built using the Django framework. It is BSD licensed and designed to provide a clean and simple base for developing e-commerce websites.

Cartridge also extends the Mezzanine content management platform. A live demo of Cartridge can be found by visiting the Mezzanine live demo.

To learn more about Cartridge, please read the Overview, which contains a features list, an installation guide, and other general information.
